# Faraway FrontEnd

这是按仓库文档 [04_28_开发文档.md](../docs/04_28_开发文档.md) 生成的 Uni-app 客户端前端工程。

当前状态：

- 已按文档生成页面结构、路由、mock API、stores、公共样式
- 页面视觉参考 `faraway/`，保留沉浸式旅行氛围、深色底、大图与圆角玻璃卡片
- 首页保留 `faraway` 风格的搜索框 + 做攻略 + 找搭子

## 运行前

当前工程使用轻量本地 store，不依赖 Pinia。

说明：

- 当前这份 HBuilderX 工程以稳定运行优先，先保留兼容性更高的写法
- 如果后续确定迁移到真正的 Vue3 uni-app 工程壳，再统一切 Pinia

## 本地后端联调

- 后端地址配置文件：`api/config.js`
- 当前实际地址以 `api/config.js` 文件内容为准
- 如果前端是在手机或模拟器里跑，通常不能用 `127.0.0.1`
- 这时请把 `api/config.js` 里的地址改成你电脑的局域网 IP，例如：`http://192.168.1.20:8000`
- 当前上传模块仍是 mock，等后端上传接口 ready 后再接真实上传
